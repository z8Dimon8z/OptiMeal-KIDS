Typography

Типографика

Заголовки должны быть округлыми и эмоциональными, основной текст - дружелюбным и читаемым. Для UI-меток используется более плотное начертание с компактной посадкой.

Заголовки

H1. Яркий детский оффер

H2. Секция с преимуществами

H3. Карточка или контентный блок

Текст

Lead. Используется для вводных абзацев, текстовых hero-блоков и пояснений рядом с CTA.

Body. Базовый текст для карточек товара, форм, списков преимуществ и контентных описаний.

Caption. Подписи, метки, статусы, вспомогательные тексты.
